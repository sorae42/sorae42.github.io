Minecraft Instances

for MultiMC, PolyMC and multiple instances supported launchers.

Please grab them here: MultiMC, PolyMC, ManyMC (for M1 Macs), UltimMC (cracked version).

Please notice that these launcher DOES NOT have Java baked with the installer unlike some of the other launchers. You will have to install Java yourself.

These Minecraft Instances are packed with performance and quality of life mods, plus some resources pack I use in Minecraft. There are no mod that add new content or whatsoever.

Those are my personal references and may contains mods/resource you might not need. Feel free to download them and make changes/disable/remove them as you want.

Mods are downloaded from Modrinth and CurseForge and are checked carefully for viruses.

The numbers of mods shown here are reported from the game itself (this includes libraries too). File size are determined from hosting services I uploaded the file on.
